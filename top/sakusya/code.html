ホーム / 作者 / アルテミジア・ジェンティレスキ

アルテミジア・ジェンティレスキ

1593年 – 1656年頃

経歴

アルテミジア・ジェンティレスキは、バロック期において最も高く評価されたイタリアの女性画家の一人です。カラヴァッジョの追随者の中で、彼女は最も才能のある一人と見なされています。当時、女性が芸術家としてキャリアを築くことは非常に困難でしたが、彼女はフィレンツェの美術アカデミーで初の女性会員となるなど、目覚ましい成功を収めました。

彼女の作品は、神話、寓話、聖書に登場する女性たちを力強く、苦悩に満ちた姿で描いているのが特徴です。「ホロフェルネスの首を斬るユディト」のような主題は、彼女の作品の中でも特に有名であり、そのドラマチックな表現と感情の深さは、彼女自身の困難な人生経験を反映していると解釈されています。
